# 清木・林ゼミ（データサイエンス学科・AI＆データベース研究室 / AIDBL）

「AI・データベース研究室（AI & Database Lab, AIDBL）」。現代の情報システムの中核をなすAIアルゴリズムとデータベースを研究対象とし、担当は清木 康（Professor）・林 康弘（Associated Professor／准教授）の2名。武蔵野大学有明キャンパス。

## 研究分野・テーマ
- 物理空間をコンピュータで計算可能な空間へ拡張する「サイバー・フィジカルスペース」を対象に、交通・観光・福祉・自然環境・教育・学習などの具体領域で課題解決と新たな価値創造を目指す。
- 研究の4つの柱（Pillars）：(1)意味計算と知識ベース、(2)エンタープライズ向け時空間・分類データマイニング、(3)自律分散協調型資源組み合わせ最適化、(4)人間・AI共創空間と潜在状況・状態推定。

## 手法・使用技術
- AI創造のための独自の「中核計算」（意味計算、潜在状況・状態推定、時空間データマイニング、時系列分析、最適化）の設計。
- 人間・AI協調のためのサイバー・フィジカルシステムアーキテクチャの設計、特定領域向け知識ベース（データ・モデル・ルール・文脈）の構築と中核計算への適用。
- データベース・SQL、数学・統計・最適化による数理モデリング、センサー・ネットワーク・サーバ・アクチュエータまでの実装。

## 活動内容・指導方針
- 1年次秋開始の「未来創造プロジェクト」で各自が研究テーマを設定し、議論と実装のサイクルを反復して深化。通常科目と並行して進める。
- 4年間の計画：1年次＝基礎構築とテーマ設定、2年次＝独自モデル開発とプロトタイピング、3年次＝モデル改良・拡大と学会発表、4年次＝卒業研究の完成・論文化と社会実装の検討。
- 既存手法をなぞらず、目的・意味・概念を哲学的に問い直して独自の中核計算を設計する探究心を重視。データサイエンス・工学・社会科学・人文知を横断する学際的アプローチを求める。

## 教育・関連
- 学部科目例：データサイエンス学、データベースデザイン、人工知能テクノロジデプロイメント、Webエンジニアリングと社会、マルチメディア知識ベース、コース演習I（人工知能クリエイション）、人間・AI協調進化システム、AI社会の情報倫理 など。
- 海外連携（ブータン王立科学技術大学、PJATK 等）や高校生向けAI大会（シンギュラリティバトルクエスト）との関わりあり。
