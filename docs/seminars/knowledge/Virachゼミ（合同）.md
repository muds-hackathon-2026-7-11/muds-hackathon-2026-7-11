# Virachゼミ（合同）（データサイエンス学部・Virach／Thatsanee／佐々木ほか6名合同）

正式名は「Virachゼミ（Virach・Thatsanee・佐々木・神崎・小林・Titi合同）」の6名合同ゼミ。AI・データサイエンスを社会課題解決や社会イノベーションへつなげる点で共通し、人間行動・感情の認識（自然言語処理・センシング）、社会イノベーションのプロトタイプ設計、時空間データの可視化・社会実装を横断的に扱う。（本要約は資料が提供されたVirach・Thatsanee・佐々木の3名分に基づく。神崎享子・小林周・Titipakorn Prakayaphunの3名は資料未提供のため未反映。）

## 研究分野・テーマ
- Virach：人間行動認識と社会イノベーション（Human Behavior Recognition）。会話・メディア分析で内容／意思／感情を把握し予測支援。自然言語処理・AI・データ分析が中核。
- Thatsanee：社会イノベーション・アプローチ（持続可能で調和した世界の形成）。データイノベーションと社会イノベーションを組み合わせ、協働デザインで社会課題を解決。
- 佐々木：デジタルビジネスイノベーション、IoT・ロボティクス。「5D World Map System」を軸にセンシング→分析・可視化→アラート配信を扱う。

## 手法・使用技術
- 自然言語処理（チャットボット、SNS感情/動向分析、教育ビデオ要約）、画像・表情認識、音声感情認識（Wav2Vec2/XLSR等）。
- IoTセンシング（介護・バイタルセンサー、MQTT、UAV＋マルチスペクトラルカメラ）、YOLOによる人体検出。
- 時空間データベース（PostgreSQL）、TF*IDFやベクトル空間モデルによるメディア相関計算、GIS可視化、IFTTTによるアラート配信、Temiロボット操作。

## 活動内容・身につくスキル
- コミュニティのニーズ評価からプロトタイプ開発・実装・インパクト評価・スケーリングまでの一連（Thatsanee）。
- センシング＋DB構築＋可視化の演習、感性・異文化マルチメディアコンピューティング、アラート送信の実証実験（佐々木）。
- 介護ロボット・AI City・スマートシティ基盤などの応用開発（Virach）。

## 体制・実績
- タイをはじめアジアの大学・機関との国際共同研究が特徴。
- 5D World Map Systemやスマートシティ基盤（AI City）などの研究・開発実績。
