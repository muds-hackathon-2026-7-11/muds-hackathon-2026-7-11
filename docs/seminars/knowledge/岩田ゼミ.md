# 岩田ゼミ（データサイエンス学科・岩田洋夫）

「未来創造プロジェクト」。データの世界を人間の身体感覚として提供するシステム "Data Sensorium" の構築に取り組むゼミ。担当は岩田洋夫教員。

## 研究分野・テーマ
- コンセプトは "Data Sensorium"（データの世界を人間の身体感覚として提供するシステム）。
- 武蔵野大学データサイエンス学部に共通する SPA コンセプト（Sensing／Processing／Actuation）に基づく。
  - Sensing：現実空間から物理的情報を抽出する（IoT、全周カメラなど）。
  - Processing：得られた情報を格納し処理する（多次元データベース等）。
  - Actuation：現実空間に物理的変化を与え、人に身体感覚を提供する。← このゼミで主に扱う領域。

## 手法・使用技術
- ロコモーションインタフェース（データの世界の中を歩く感覚を与える。5号館6階の岩田ブースに設置）。
- ハプティックインタフェース（データに「手応え」を与える）。
- 作業環境として、ロボットキット mBot2、各種センサー・アクチュエータを使用。必要に応じて VRゴーグル（Meta Quest 2）を提供。

## 活動内容・進め方
- 自分の興味に合わせて、どのような体験をデザインするかを策定する。
- 実装に向けてシステム設計を行う。
- 使用可能なハードウェアを用いて、プロトタイプの実装を行う。

## 成果物目標
- 発表会で体験可能なシステムを作成する。
- その体験をデータ化する。
