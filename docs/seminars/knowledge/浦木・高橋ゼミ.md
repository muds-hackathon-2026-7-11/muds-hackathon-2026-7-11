# 浦木・高橋ゼミ（データサイエンス学部・浦木麻子／高橋雄介）

「データサイエンスの視点から人間の進化・存続に貢献」を掲げる浦木・高橋の合同ゼミ。自然現象を人間がデータ（離散値）として分析・議論する営みを人間の経験や知識の宝の山と捉え、データベースで保証される根拠にAIの創造性を適切に組み合わせることで、人間へ新しい行動支援を発信し、人間の進化・存続に貢献することを目指す。

## 担当教員（専門）
- 浦木 麻子（准教授・博士（政策・メディア））：時系列データの意味的理解が専門。
- 高橋 雄介（准教授・博士（政策・メディア））：知識ベースとその応用、グロースハック、顧客開発、プロダクトデザインが専門。

## 研究分野・テーマ
- 研究テーマ（キーワード）：Semantic Data Scope／Database／Data Mining／Conceptual Modeling／Time-series Data Modeling／Semantic Data Modeling。
- 浦木テーマ：時系列データの意味的理解。人間が状況分析で行う「着目点の移動」「解像度のコントロール」といった人間らしい時系列データの意味的な理解・比較・気づきの機能プロセスのモデル化を模索し、応用分野を議論する。
- 高橋テーマ：データをプロダクト・グロースの行動指針（actionable metrics）として活用する。

## 活動内容・指導方針
- 研究活動とは、新しいアイディア・方法論・計算式等を生み出し、社会に発信・活用されるよう進めていく活動。自身の研究テーマに沿って新規性を学術的に深め、国内外への発信（研究発表）と社会実装を目指す。
- 「いつでもどこでも観察→小さなアイディア思いつく→メモ」という毎日のルーティーンを重視。日々身の回りの現象を観察し、「人間ならこれが計算できるのに」というポイントを見つける洞察力を大切にする。得意なこと・興味・問題意識の高いところに着目すると効果的。
- 金曜日のゼミはオンサイト・オンライン両方を組み合わせて実施する。
