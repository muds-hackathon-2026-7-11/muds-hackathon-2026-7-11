# 石橋・槌屋ゼミ（データサイエンス学部・石橋直樹／槌屋洋亮）

「未来創造プロジェクト／ビジネスマーケティングプロジェクト "Governance Design"」。ITを活用して社会課題を解決するため、Web環境でのモノ作り（開発）から実社会での運用・実験・起業までを一貫して取り組むプロジェクト。担当は石橋直樹・槌屋洋亮の2教員（配布資料は石橋教員名義）。

## 研究分野・テーマ
- Governance Design＝政治・経済・社会に対する工学的働きかけ。新しい仕組み・枠組み(System)の設計(Design)・構築(Implementation)・運用(Application)と、実験(Experimentation)・評価(Evaluation)を伴う一連のプロセス。
- テーマは「ITを活用した社会課題解決の検討」。SDGsに代表される国際的な社会課題を、WebやITの活用で解決する余地を探る。

## 手法・使用技術（Web環境におけるモノ作り）
- クライアント側：HTML／CSS／JavaScript(AJAX, jQuery)、Objective-C(iOS)／Java(Android)。
- サーバ側：CGI(Perl, Ruby, C)／Java(JSP)／PHP。
- データ保持：File System／PostgreSQL／MySQL／NoSQL。サーバOS：Windows／macOS／Linux 等。

## 活動の考え方・身につくこと
- Webはヒトとヒト・モノ・コトを繋ぐ媒体（ヒト＝SNS・Recruiting／モノ＝E-Commerce・IoT／コト＝Museum・e-Learning・Game・Crowd Funding 等）。
- 作り方の習得に加え、「ヒトと何をどう繋ぐのか?」というアイディアの設定を重視。
- 期待されるOutput：サービス開発、実社会での運用・実験、起業。

## 担当教員
- 石橋直樹：博士(政策・メディア)、株式会社Governance Design Laboratory代表取締役。データベース／マルチデータベース／マルチメディアシステムの研究開発に従事。
