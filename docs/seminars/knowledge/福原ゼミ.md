# 福原ゼミ（データサイエンス学科・福原義久）

「ビジネスマーケティング／IoT・ロボティクス プロジェクト」。担当の福原義久教員はフルスタックエンジニアで、機械学習・深層学習の講義を担当し、スタートアップを立ち上げて経営（AIトレーニングマシンを開発）。

## 研究分野・テーマ
- IoT・ロボティクス（IoTガジェットクリエイション）と、ビジネスマーケティング（ビジネスモデル構築）の2本柱。
- 上記に当てはまらないテーマも制約なく自由に研究可能。「自分が面白いと感じるテーマを見つける」「ユーザーも共感できる成果物を目指す」を重視。

## 手法・使用技術
- IoTデバイス（Raspberry Pi、Arduino）、各種センサー・カメラ・アクチュエータ、スマートロボットカーなどの組み合わせによる問題解決。
- 機械学習・深層学習・深層強化学習、物体検出、画像・音・センサーデータの分析、Webサービス開発。

## 活動内容・身につくスキル
- 作りたいモノ／解決したいことに対し、どのデバイスを使い・どう実装するかの技術指導。ガジェットやWebサービスのプロトタイプ作成をサポート。
- 成果物目標：IoT・ロボティクス系は論文・特許、ビジネス系はピッチコンテスト・ビジネスアイデアコンテスト。

## 過去の研究テーマ・実績
- 例：脳波センサーによるIoT/ロボット操作、歌詞・楽曲に連動したステージ照明制御、機械学習によるスポーツ動作分析、空間の印象に基づく香り／アロマのレコメンド、メイクアップ手法の提案 など。
- 卒業論文・学会発表の実績が豊富（情報処理学会全国大会、国際会議 IIAI 等での発表多数）。
