# 長谷川ゼミ（長谷川 理）

「人の学びとコンピュータの学びの違い」を出発点に、サイバー空間と物理空間が融合する中で教育と学びがどう変化するかを探究するゼミ。認知負荷のモデル化と、xR環境での学習支援の2本柱を掲げる。

## 研究分野・テーマ（Research Question）
- 人の学びの仕組みとコンピュータの学びの仕組みの違いは何か。
- 人とコンピュータ、サイバー空間と物理空間が融合していく中で、教育と学びはどのように変化していくのか。

## アプローチ（2本柱）
- サイバー空間と物理空間が融合した環境での教育・学びにおける認知負荷の分析・モデル化。キーワードは「Multi Fusion Cognitive Model」。関連概念：認知心理学、認知負荷モデル、身体知、興味・関心推定、視線、生体情報（脳波など）、体軸、姿勢角 など。
- xR（VR・AR・MR・SR など）環境での最適な学習サービスや学習コンテンツの検討。キーワードは「Socrates's Eye」。関連概念：先進的学修支援、マルチエージェント、学習者モデル、教授モデル、ホログラム教材 など。

## 手法・使用技術（Skill Set）
- Unity、C#、Objective-C、Java、Python、HTML、CSS、RDBMS、ML（機械学習）、DL（深層学習）、3D Modeling など。
- 実装環境：HoloLens、Apple Vision Pro、iPhone などのデバイスを利用（資料の写真より）。

## 活動内容
- ラーニングアクティビティ・コミュニケーション・表情・生体情報などの入力から認知負荷モデル（Fusion Cognitive Model）を構築。
- ホログラムや拡張現実を用いた学習コンテンツ・学習サービスの検討。
- 「コンピュータと共に成長する融合環境での学びのデザイン」として、人（学習者）と複数のAIエージェントが強化学習的な枠組みで相互に学び合う構図を扱う。
