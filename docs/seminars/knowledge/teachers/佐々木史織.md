# 佐々木 史織（Shiori Sasaki）（データサイエンス学部）

「デジタルビジネスイノベーション、IoT, ロボティクス（Digital Business Innovation, IoT-Robotics）」。センシング（IoT）→プロセッシング（分析・可視化）→アクチュエーション（配信・アラート）の一連を扱う「5D World Map System」を軸に、感性・異文化マルチメディアコンピューティングと時空間データ解析を組み合わせ、社会実装（Social Implementation／SDGs）を目指すプロジェクト。（本ファイルはSasaki.pdf単体の内容に基づく。）

## プロジェクト型実習・実験の中核：5D World Map System
- センシング入力：センサーデータ、統計データ、地理・空間データ、テキスト／画像／動画／音声データ、衛星写真、小型UAV＋マルチスペクトラルカメラ。
- プロセッシング（分析・可視化）：(1)リアルタイムマッピング、(2)変化量・差分・類似性計量・探知、(3)分析的可視化、時空間データ可視化、マルチメディア検索。
- 保持DB：メディアDB、ユーザDB、地理・空間DB、センサーDB。
- アクチュエーション：(4)リアルタイム配信、アラート配信。
- 位置づけ（UN ESCAP文脈）：Data／Statistics／Knowledge Sharingを核に、有効なデータと分析ツールの提供、データギャップの解消、エビデンスに基づく政策決定支援（Decision Support／Policy Makers）。

## 関連する技術要素（S-P-A）
- IoT（Sensing）：リアルタイム・センシング（Small Sensor - 5D）、センサーデータ・アノテーション（Data Labelling）。
- Multimedia（Processing）：自然言語処理・テキストマイニング、TF*IDF、画像処理・画像DB、ベクトル空間モデルによるメディア相関量計算、時空間データベース構築（PostgreSQL）。
- Visualization（Actuation）：地理情報データ可視化（GIS）、Warning Messaging（5D-IFTTT）によるプッシュ系メッセージ送信。
- 全体像：Cyber Space×Real SpaceのSensing-Processing-Actuationループ。

## 演習・応用研究
- センシング＋DB構築＋可視化の演習、感性・異文化マルチメディアコンピューティング、5Dアラート送信の実証実験、時空間・差分計量データ可視化演習。
- Web Service連携：IFTTT経由でGmail／LINE／WhatsApp／Telegramへ通知。IoTツール（Raspberry Pi、ドローン、各種センサー）を活用。
- 応用研究テーマ例：世界のニュース、絶滅危惧種、国際テロ、環境異常検知、ヘルスデータ、世界遺産・観光、世界のアート・工芸・音楽・料理、感性メディア生成 など。
