# 渡邊・糸田ゼミ（渡邊紀文・糸田孝太／未来創造プロジェクト）

「人を理解するためのデータサイエンス」を掲げる合同プロジェクト。ロボット・エージェント・神経回路モデル・LLMなどデータサイエンス技術を使って人をモデル化し理解する。渡邊紀文（知覚情報処理・知能ロボティクス・認知科学）と糸田孝太（人工知能・社会的認知科学・MAS）が担当。

## 研究分野・テーマ
- 人の意図推定と協調するロボット、集団行動を獲得するエージェント、視覚的注意の神経回路モデルと行動誘導、対話型プログラミング学習支援（LLM）、Pepperを使った支援サービスなど。
- その他：脳をモデルとした情報処理モデル、IoTによる環境センシング（CO2）、ロボカップ（サッカーシミュレーション、@Home）。

## 手法・使用技術
- 強化学習、多クラス分類、スパイクタイミングニューラルネットワーク、エージェントモデル、因果性分析、LLM/VLM、ChatGPT。
- 実験機材：Pepper、ドローン、視線計測ヘッドマウントディスプレイ、モーションキャプチャ、アクションカメラ。
- 開発：Webインタフェース（TypeScript, React）、AIアシスタント Cline など。

## 活動内容・進め方
- 授業開始前に実施目標・活動計画を提出し、各自のテーマで研究を進める。毎週「研究ノート」を作成・提出（進捗管理と最終成果物）。
- 定例ミーティングは金曜3・4限。プロジェクト全体での情報共有、進捗報告（各学期1〜2回）、教員・TAによる個別相談、中間・最終成果発表会。
- 任意参加の機械学習勉強会（毎週金曜2限オンライン、LLM/VLMの最新動向・論文紹介）。

## 成果物・実績
- 2025年度卒論例：AIMEを用いた睡眠の質改善システム、ドラム演奏のリズムずれフィードバック、LLM×拡散モデルによる表データ補完、非言語情報を用いた適応型チャットボット。
- 外部発表：画像電子学会（群衆人数推定）、スポーツデータサイエンスコンペ（野球の「流れ」）、DEIM2026（漫才の「間」分類）など。国内学会・国際会議（IJCNN, ICAART, HAI, BICA, SMC 等）での発表実績。
