## プロジェクトのコンセプト：Data Sensorium

**"Data Sensorium"** = データの世界を人間の身体感覚として提供するシステム。

武蔵野大学データサイエンス学部に共通する **SPA コンセプト**（Sensing / Processing / Actuation）に基づき、サイバー空間と現実空間を橋渡しする。

### SPA の流れ
- **Sensing**：現実空間から物理的情報を抽出する（IoT、全周カメラ など）。
- **Processing**：得られた情報を格納し処理する（多次元データベース 等）。
- **Actuation**：現実空間に物理的変化を与える（人に身体感覚を提供する）。
  - ⇒ **このゼミでやること**は主にこの Actuation の領域。

### 提供する体験（インタフェース）
- **ロコモーションインタフェース**：データの世界の中を歩く感覚を与える（5号館6階 岩田ブースに設置）。
- **ハプティックインタフェース**：データに「手応え」を与える。

### ゴール

### 作業内容

- 使用可能なハードウェアを用いて、プロトタイプの実装を行う。
