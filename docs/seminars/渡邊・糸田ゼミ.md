# 渡邊・糸田ゼミ（未来創造プロジェクト）

- **プロジェクト名**：2026 未来創造プロジェクト「人を理解するためのデータサイエンス」
- **担当教員**：渡邊 紀文（わたなべ のりふみ）・糸田 孝太（いとだ こうた）
- **出典**：ゼミ紹介資料（`Watanabe_Itoda.pdf`, 全18ページ／渡邊・糸田 合同資料）

## 担当教員（専門）
- **渡邊 紀文（准教授）**：知覚情報処理、知能ロボティクス、認知科学。担当授業はロボティクス・IoT／機械学習アルゴリズムデザイン／未来創造プロジェクト。
- **糸田 孝太（講師）**：人工知能、社会的認知科学、MAS（マルチエージェントシステム）。担当授業は未来創造プロジェクト／機械学習活用／人工知能実践プロジェクト（AI副専攻）。

## プロジェクトの概要
ロボット、エージェント、神経回路モデル、LLM などのデータサイエンスに関わる技術を利用して、**人をモデル化し、理解する**ことがテーマ。

## 研究テーマ例

### テーマ例1：人の意図を推定して協調するロボットの開発
- 人に対して受動的に行動するだけでなく、人の意図に合わせて能動・受動の戦略を切り替える（**メタ戦略**）を持つロボットの開発。ロボットが能動的に働きかける、人とロボットの新しい協調関係を構築する。
- モーションキャプチャを利用した人と仮想ロボットの協調実験、グリッドワールドで人の能動・受動戦略を獲得するエージェント。
- キーワード：メタ戦略モデル、モーションキャプチャ、強化学習、LLMエージェント。

### テーマ例2：集団行動を獲得するエージェントモデルの構築
- 相互に意図を共有し協調行動をするエージェントモデルを構築し、人の集団行動における行動決定過程を明らかにする。集団行動の形成に必要な視線やジェスチャなどの身体情報を抽出する。
- サッカーを対象とした人の集団行動の分析、意図共有モデルとエージェントを利用した集団行動のシミュレーション。
- キーワード：多クラス分類モデル、行動の分節化、因果性分析、一人称視点、エージェントモデル。

### テーマ例3：視覚的注意の神経回路モデルと行動誘導
- 人がどのように外界に注意を向け行動しているのかを、脳の注意機構のモデル化および注意を制御する視覚刺激を用いた行動の創発によって明らかにする。視覚刺激により意識下で行動を誘導し、安全安心に行動を制御する。
- 視覚的注意時の神経活動を再現する神経回路モデル、視覚誘導自己運動感覚を提示する装置とパーソナルモビリティ搭乗者の誘導。
- キーワード：脳の視覚野のモデル、スパイクタイミングニューラルネットワーク、視覚誘導自己運動感覚、周辺視ディスプレイ、パーソナルモビリティ。

### テーマ例4：対話型プログラミング学習支援システムの開発
- LLM を利用した新しい学習支援システムの構築。プログラミング学習での教員と学生の対話構造の分析から、理解度を元にしたパーソナルな学習支援を行う。Webインタフェースを用いたプログラミング作成状況の観測と学習ヒントの提示。
- プログラミングの授業で利用しているチャットボットと、VS Code で利用できる AI アシスタント "Cline"。
- キーワード：パーソナルAI、対話構造分析、マルチターン型対話、LLM、Webインタフェース（TypeScript, React）。

### テーマ例5：Pepper を使った人を支援するコミュニケーションサービス
- ロボットプログラミングで人を支援する新しいサービスを構築。表情や音声、対話内容からユーザの意図を推定し能動的に働きかけるロボット。ロボットを利用しないとできない新しいサービスを提案し、実装・評価する。
- シミュレータによる Pepper のアプリケーション開発、夜道の不安解消やストレスチェックをする Pepper。
- キーワード：ロボットプログラミング、ヒューマンロボットインタラクション、サービスデザイン、マルチモーダルAI、ChatGPT。

### その他の研究テーマ
- **脳をモデルとした情報処理モデルの構築**：神経細胞間の距離に依存した結合の学習と切り替えによる視覚の結びつけ問題・語の多義性解消モデル、ボトムアップとトップダウンの情報統合による視覚情報再構成モデル。
- **IoTデバイスを用いた環境センシング**：二酸化炭素の多点同時センシングによる濃度予測モデル、聴覚・体性感覚の制御による呼吸誘導での CO2 濃度変化の分析。
- **ロボカップ**：帰納論理プログラミングを用いたサッカーシミュレーションの戦術パターン抽出、Jリーグ選手のパス行動分析に基づくパス行動生成、ロボカップ@Home のタスク開発。

## プロジェクトで使える道具
- **Pepper**：カメラで人を認識、マイク・スピーカーで音声対話、頭や腕へのタッチによる身体インタラクション、タッチパッド表示、ChatGPT を利用した対話。
- **ドローン**：人の行動を俯瞰的に撮影。
- **視線計測可能なヘッドマウントディスプレイ**：バーチャル空間の提示、視線分析。
- **モーションキャプチャ**：人の動きの詳細分析。
- **アクションカメラ**：体に装着して動作を分析。

## 進め方・授業構成
- **研究テーマ**：授業開始前に設定したテーマと計画に基づき、各自で研究を進める。
- **提出物**：授業開始前に実施目標・活動計画・学習内容を提出。毎週、進捗を記入した**研究ノート**を提出（進捗管理と最終成果物）。研究ノートにはプロジェクトで利用する理論・手順（文献・調査・データ収集・ツール設定・分析手順）と成果（作成プログラム、グラフ・表、概念図）を含む。
- **定例ミーティング**：金曜3・4限。プロジェクト全体での情報共有（おすすめツール・サイト・文献、メンバーへの相談やアンケート、就活相談）、進捗報告（各学期1〜2回）、個別相談。
- **成果発表**：プロジェクト内での中間成果発表・最終成果発表。個別相談は研究ノートを元に教員および TA が対応。

## ゼミ選びの観点まとめ
- **向いている人**：ロボット・エージェント・神経回路・LLM などの技術で「人」を理解・モデル化することに興味がある人、Pepper やモーションキャプチャ・HMD など多様な機材で手を動かしたい人、人と機械の協調・インタラクションに関心がある人。
- **キーワード**：人のモデル化、エージェント、強化学習、ヒューマンロボットインタラクション、認知科学、LLM、Pepper、モーションキャプチャ、集団行動、神経回路モデル。
