# ゼミ紹介資料の要約ドキュメント

各ゼミの配布資料（教員提供のPDF）を要約したドキュメント集。マッチ度診断・相談チャットの知識ソース（`seminars.knowledge`）や、人による比較検討に用いる。

## 構成
| パス | 用途 |
| --- | --- |
| `<ゼミ名>.md` | 人間が読む詳細版。研究テーマ・手法・機材・卒論例などを網羅。 |
| `knowledge/<ゼミ名>.md` | AI／DB投入用の簡潔な要約（`seminars.knowledge` 向け・約600〜800字）。 |
| `knowledge/teachers/<教員名>.md` | 合同ゼミで教員ごとに資料が分かれている場合の、教員単位の要約。 |

## 作成方針
- **配布資料に記載された事実のみ**で作成し、資料に無い情報は推測・補完・創作していない。
- ゼミ選び／マッチングに不要な情報（連絡先・URL・提携先ロゴ・書誌情報・教員の個人経歴など）は除去済み。
- 元PDFは私有資料のため `/.private/` にあり、リポジトリには含めない。
